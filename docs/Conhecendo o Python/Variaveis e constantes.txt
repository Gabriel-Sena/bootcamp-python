Variáveis: valores que podem sofrer alterações ao decorrer da execução;
Para alterar um valor só basta atribuir um novo valor a variável.

Constantes: possui o valor fixo até o final da execução, o valor é imutável.

Python não tem constante, não existe uma palavra reservada para informar que o valor é constante. Para isso se usa a convenção (tratado em comum), onde se deve criar a variável com o nome todo em maíusculo.
Variável escrita toda em maíusculo = constante.


Boas práicas
-Padrão de nomes em snake case(uso de _);
-Nomes sugestivos;
-Nomes de constante em maíusculo.
