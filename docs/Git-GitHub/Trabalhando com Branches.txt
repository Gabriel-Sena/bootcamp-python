Branch: é uma ramificação do projeto.

-Criando e mexendo com branchs
    git checkout -b (nome da branch): cria um branch e troca da branch que eu estou para a branch que eu mecionei no comando;
    git checkout (nome da branch): redireciona para a branch mencionada;
    git branch -v: lista o útimo commit de cada branch;
    git merge (nome da branch que quero mesclar): mescla a branch que mencionei com a branch que estou;
    git branch: lista as branches que tenho, o "*" sinaliza a branch que estou no momento;
    git branch -d (nome da branch): serve para deletar a branch que mencionei no comando;
